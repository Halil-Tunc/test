# Pflugerville Community Resources

A community resource hub designed to help Pflugerville residents find local services, support, and opportunities more easily and efficiently.

## Overview
This project was built to make community resources more accessible in one place. The goal is to help users quickly locate the information and support they need without having to search across multiple websites or sources.

## Purpose
The purpose of this platform is to:
- improve access to community resources
- make information easier to find
- support residents in Pflugerville through a more organized and user-friendly experience

## Tech Stack
- React
- TypeScript
- React Router
- Leaflet
- React Leaflet
- Vercel

## Features
- interactive resource discovery
- map-based functionality
- organized access to community information
- user-friendly web interface
